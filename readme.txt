An easy and advanced Sugar webform generator for Wordpress to capture leads. WP Sugar is also a WordPress Sugar Integration that syncs your Wordpress members to Sugar Contacts.

== Description ==

WP-Sugar plugin is a simple, advanced, easy to one time setup Sugar webforms generator for Wordpress. 

- No HTML 
- No Manual Work 
- Simple Setup
- User friendly UI
- More Options 
- Short Code Featured
- Dynamically Generated
- Add, Edit fields at any time 
- Easy Manageable 

It helps to generate a webforms in very simple steps and few clicks via user friendly plugin interface as shortcode.  

1. Install and configure you sugar crm details
2. Go and click Fetch CRM fields button
3. Enable the required fileds for the capture form
4. Save and use the short code to publish a capture form

Short code can be used in page, post or widget as a contact/capture form to convert your traffic/visitors as high potential leads to Sugar CRM. No need to create, configure, copy and paste form html from sugar crm to your website. This involves lot of manual work and time. Rather WP Sugar advanced Sugar webforms generator makes this easy in matter of time as short code directly in plugin interface without switching to Sugar. 

Pro Version Features
- Unlimited Forms with different field set
- Advanced controls like change position order, make manadatory and label name etc
- Simple Shortcode manager
- Assign User for captured leads / contacts

== Installation ==

1.	WP Sugar  is very easy to install like as other simple Wordpress plugins. No need to edit or modify anything here in code.
2.	Unzip the file 'wp-sugar.zip'.                              
3.	Upload the ' wp-sugar' directory to '/wp-content/plugins/' directory using ftp client
4.	Or you can upload and install wp-sugar.zip through plugin install wizard in WP admin panel.
5.	Activate the plugin through the 'Plugins' menu in Wordpress.
6.	After activating, you will see an option for ‘Wp sugar’ in the admin menu (left navigation).
